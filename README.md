<h2>Brief introduction:</h2>
<p>This is a multilingual web application that allows users with different roles (professors, coordinators, department heads, scholarship agents) to manage attendance. The administrator can manage permissions and subjects using PHP, HTML, CSS, JavaScript, and jQuery.
</p>
